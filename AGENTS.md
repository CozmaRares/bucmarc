## Agent skills

### Issue tracker

Issues and PRDs for this repo live as markdown files under `.scratch/`. QA work should go under `.scratch/qa/<slug>/`. See `docs/agents/issue-tracker.md`.

### Triage labels

The canonical triage roles use the default label strings in this repo. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout with a root `CONTEXT.md` and `docs/adr/`. See `docs/agents/domain.md`.
